    aix-errlog¶ ↑

    Ruby gem for interfacing with AIX's error logging facilities. Uses ruby ffi. For documentation see taywee.github.io/aix-errlog/ or www.rubydoc.info/gems/aix-errlog/

    Installable exactly how you'd expect from ruby gems:

    gem install aix-errlog
